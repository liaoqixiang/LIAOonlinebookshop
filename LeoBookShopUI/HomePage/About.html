Technology and How to Use

Technology:

DataBase: MSSQL

Access to Database FrameWork: Ado.Net(My second project use EF ORM)

Front-end: Jquery(to ajaxifyling my web App) and Bootstrap

BackEnd: c#.Net, Asp.Net WebForm, using Generic handler to deal with authentication and jquery ajax since i dont need to show anything, ashx is a more efficient way of implementing it accroding to Asp.Net Request Flow.

Not using any Controls in my project since original web development gives me more flexibility and expansibility.

Although webform aspx has tight coupling, it is much more convenient than Asp.Net MVC. I also use static html page to store detail of books which can release a lot of server side pressure.

How to Use:

This is a web app selling books. so the main functionality is in bookshop part(as you can see in the nav bar.) Account: you can register and sign in to add the books into your own shopping cart(anonymous user cannot have this access(it's more complex.))

BookShop: you can view all books with pagination.

you can order the price of book ascending or descending.

you can choose category of books

you can go to the detail page of the book and comment it.

Simply clicking add to cart, you will be able to add it into your shopping cart and view it in cart(nav bar).

you can modify the amount of books you want to buy and you will have the total cost of books.

you can pay as well.
